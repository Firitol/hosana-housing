# **App Name**: Hosana Nexus

## Core Features:

- Secure Authentication & Authorization: Implement robust login and registration, along with role-based access control (Mayor, Sub-city, Woreda, Kebele) to ensure users have appropriate permissions and dedicated dashboard views.
- Housing Data Management: Enable full CRUD operations for housing records, capturing details such as householder name, phone, family size, house type, full address, and GPS coordinates within the Ethiopian administrative hierarchy.
- Dynamic Search & Filtering System: Provide a powerful search engine allowing queries by house number or householder name, complemented by filters for Sub-city, Woreda, and Kebele, with results displayed in a responsive, sortable, and paginated table.
- Interactive Google Map Integration: Visualize housing locations accurately on an embedded Google Map using stored GPS coordinates, allow selection of locations directly from the map during data entry, and highlight all registered houses.
- Role-Specific Dashboards: Present tailored dashboards for each user role, offering clear overviews of total registered houses, distribution by administrative division, recently added records, and essential statistics with simple charts.
- Comprehensive Audit Trails: Maintain detailed audit logs tracking all user activities related to housing record creation, updates, and deletions, providing a clear history of data modifications.
- AI-Powered Administrative Briefs: Utilize an AI tool to generate concise, human-readable summaries from the dashboard statistics and audit logs, providing high-level officials with quick insights into housing trends and system activity.

## Style Guidelines:

- Primary color: A deep, professional blue (#294499) that conveys trust and stability, serving as a key branding element and for interactive components.
- Background color: A very light, desaturated blue (#ECEFF7) providing a clean and serene canvas for the content, enhancing readability and reducing eye strain.
- Accent color: A vibrant blue-green (#19C2E0) analogous to the primary, used sparingly for calls to action, important notifications, and highlighting key information to create visual interest and guidance.
- Headline and body font: 'Inter' (sans-serif) for its modern, clean aesthetic and excellent legibility across various screen sizes, making it ideal for data-heavy administrative dashboards.
- Utilize a consistent set of clean, vector-based icons that are easily recognizable and reinforce the functionality of buttons and navigation elements. Prioritize clarity and simplicity.
- Employ a responsive grid-based layout with generous white space to ensure optimal readability and user experience on both desktop and mobile devices. Use card-like structures for distinct data sections on dashboards.
- Implement subtle, functional animations for state changes (e.g., loading spinners, form validation feedback, tab transitions) to provide smooth user feedback without distracting from the core administrative tasks.
